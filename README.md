## Project Setup

### Prerequisite
* Ruby 2.6.3 or above;
* Rails 6.0.2

### Install

> Install Necessary Packages.
```sh
bundle install
```

> Create Database.
```sh
rails db:migrate
```

> Add Sample Data.
```sh
rails db:seed
```

### Run Server

```sh
rails server
```

### Run tests

```sh
rspec
```
